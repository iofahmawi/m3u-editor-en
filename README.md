# 📺 M3U/M3U8 Playlist Editor

A comprehensive and advanced web tool designed to simplify managing and editing IPTV playlist files. Built entirely with HTML, CSS, and JavaScript, it runs directly in the browser with no installation required — ensuring easy access and complete privacy, as your data never leaves your device.

### 🔗 Live Demo
* [🚀 Live Preview](https://iofahmawi.github.io/m3u-editor-en)

---

### ✨ Key Features

* **Create & Load Playlists**: Upload files (`M3U`, `M3U8`, `PLS`) via drag and drop, load playlists from multiple URLs, or start a new playlist from scratch.
* **Full Channel Management**: Complete control over channels — add, edit, delete, and reorder via drag and drop, with instant search and filtering.
* **Channel Status Checker**: A powerful tool to verify whether each channel is working or not, with the ability to delete all broken channels in one click.
* **Bulk Actions**: Select multiple channels and perform advanced operations at once — alphabetical sorting, moving, group editing, or exporting.
* **Advanced Find & Replace**: Search for specific text and replace it across channel names, groups, or stream URLs — for all channels or a selected subset.
* **Professional Cleanup Tools**: Remove duplicate channels with flexible criteria, sort the full list alphabetically, and manage groups (rename, delete, merge).
* **Professional Export & Save**: Export the final playlist in multiple formats (`M3U`, `M3U8`, `PLS`, `JSON`, `CSV`, `Enigma2`), with the option to export only selected channels.
* **User-Friendly Interface**: A modern, responsive design that works efficiently on desktop and mobile, with interactive notifications and dark mode support.

---

### 🛠 Technologies Used

* **HTML5, CSS3 & Vanilla JavaScript**: No external frameworks.
* **JSZip**: For file compression and playlist export.
* **HLS.js**: For HLS stream playback support when checking channel status.
* **Plyr.js**: For the built-in video player in the channel checker.
* **Service Worker**: For PWA support and offline functionality.

---

### 🚀 How to Use

1. **Get Started**: Drag and drop your file, load from URLs, or create a new playlist.
2. **Edit the Playlist**: Use the features available in the main toolbar.
3. **Advanced Tools**: Access additional and advanced options from the "Additional Options" panel.
4. **Save Results**: When done, click "Export Playlist", choose your preferred format, and download the file.

---

### 📂 Project Files

* `index.html`: The main file containing the UI and all program logic.
* `manifest.json`: App manifest file for device installation (PWA).
* `sw.js`: Service Worker file for offline support.
